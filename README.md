# Feedback System
A simple feedback system that allows users to submit feedback and stores it for review by the support team.

## Usage
1. Create an instance of the `FeedbackSystem` class.
2. Use the `submit_feedback` method to submit feedback.
3. Use the `get_feedback` method to retrieve a list of submitted feedback.
4. Use the `save_to_file` method to save the feedback to a file.
5. Use the `load_from_file` method to load the feedback from a file.

## Testing
Run the tests using `python -m pytest` in the `tests` directory.
