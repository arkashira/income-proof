# Business Model Canvas – **income‑proof**

| **Building Block** | **Details** |
|--------------------|-------------|
| **Value Proposition** | • **Instant, secure QuickBooks data import** – one‑click OAuth authentication and automated extraction of income‑related transactions.<br>• **Verified income proof** – normalized JSON output ready for downstream risk‑assessment, lending, or payroll workflows.<br>• **Compliance‑first** – data encrypted at rest & in transit, SOC 2‑ready, and GDPR‑compatible.<br>• **Developer‑friendly** – lightweight Python SDK, clear API docs, and CI‑tested library that can be embedded in any SaaS product. |
| **Customer Segments** | 1. **FinTech lenders & credit‑scoring platforms** that need reliable borrower income data.<br>2. **Gig‑economy marketplaces** (rideshare, delivery, freelance) that must verify contractor earnings for payouts.<br>3. **Payroll & HR SaaS** providers seeking automated income verification for onboarding.<br>4. **Accounting firms & SMBs** that want a simple export of QuickBooks data into custom analytics tools. |
| **Channels** | • **QuickBooks App Marketplace** – listed as a certified integration.<br>• **Direct sales** – outbound outreach to fintech & payroll SaaS decision‑makers.<br>• **Developer portal** – public docs, GitHub repo, and sample code to drive self‑service adoption.<br>• **Partner integrations** – bundled with Axentx’s existing data‑harvest pipelines and other Axentx products.<br>• **Webinars & technical workshops** – showcase security & compliance features. |
| **Revenue Streams** | 1. **Subscription SaaS** – tiered plans (Starter, Professional, Enterprise) based on monthly API call volume.<br>2. **Usage‑based pricing** – $0.001 per imported transaction after free tier.<br>3. **Professional services** – custom integration, data mapping, and compliance consulting.<br>4. **Marketplace revenue share** – 15 % of QuickBooks‑partner referral fees (if applicable). |
| **Cost Structure** | • **Cloud infrastructure** – compute, storage (encrypted S3/Blob), and networking for the ingestion service.<br>• **QuickBooks API licensing** – per‑developer account fees & potential per‑call costs.<br>• **Security & compliance** – SOC 2 audit, encryption key management, vulnerability scanning.<br>• **Engineering & QA** – salaries for dev, QA, and DevOps (including test automation with pytest).<br>• **Customer success & support** – tier‑based support staff and SLA tooling.<br>• **Marketing & partner acquisition** – marketplace fees, webinars, and sales enablement. |
| **Key Resources** | • **`income-proof` codebase** – Python SDK, OAuth flow, data normalization logic.<br>• **Axentx BRAIN (pgvector) knowledge store** – reusable patterns for secure data handling and API design.<br>• **Intuit QuickBooks API credentials** & certified developer partnership.<br>• **Cloud environment** – secure containers, encrypted storage, CI/CD pipelines (GitHub Actions).<br>• **Team expertise** – senior backend engineers, security engineers, compliance officer, product manager. |
| **Key Activities** | 1. **Develop & maintain** the QuickBooks integration (OAuth, data extraction, schema mapping).<br>2. **Secure data handling** – encryption, access controls, audit logging.<br>3. **Compliance monitoring** – SOC 2, GDPR, CCPA checks and regular audits.<br>4. **API version management** – stay aligned with QuickBooks API updates.<br>5. **Customer onboarding & support** – SDK docs, sample apps, troubleshooting.<br>6. **Performance & reliability ops** – monitoring, auto‑scaling, incident response. |
| **Key Partners** | • **Intuit (QuickBooks)** – API access, certification, co‑marketing.<br>• **Cloud providers** – AWS, Azure, or GCP for secure compute & storage.<br>• **Security auditors** – third‑party SOC 2 and penetration‑testing firms.<br>• **Axentx internal teams** – BRAIN for knowledge reuse, QA for test automation, Revenue‑validation for market fit.<br>• **Payment processors** – Stripe/PayPal for subscription billing. |
